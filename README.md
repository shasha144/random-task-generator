## Описание программы

Random Task Generator — это графическое приложение для генерации случайных задач. Программа позволяет получать случайную задачу из предопределённого списка, просматривать историю сгенерированных задач, фильтровать их по типу (учёба, спорт, работа), добавлять новые задачи и сохранять историю в JSON-файл.
